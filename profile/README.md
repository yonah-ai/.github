## Repositories

- **dove-edu** — education vertical
  - [dove-edu-backend](https://github.com/dove-agent/dove-edu-backend) — Chalice REST + WebSocket API and CrewAI worker (Apache-2.0)
  - [dove-edu-frontend](https://github.com/dove-agent/dove-edu-frontend) — React + Vite frontend (Apache-2.0)

Additional verticals may follow as academic work on the same architectural pattern.

## About

Dove Agent is **academic open-source work**, accompanying peer-reviewed papers in the author's research portfolio. It has **no commercial purpose**.

The name **Dove** is chosen in tribute to the Holy Spirit.

> *"I saw the Spirit descending from heaven like a dove, and it remained on him."*
> — John 1:32

The Greek word translated *remained* is μένω — the same verb the Gospel of John uses for *dwell* and *abide*. The Spirit's coming and its dwelling are one motion. This project carries the name in that spirit: a quiet, working presence that remains with the work it is given.
